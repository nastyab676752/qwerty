Требования
Python 3.x
Библиотеки: requests, tkinter (обычно входит в стандартную поставку Python)
Как получить API-ключ
Перейдите на сайт exchangerate-api.com.
Зарегистрируйтесь.
Войдите в свой аккаунт и создайте API-ключ в разделе "API Keys".
В коде замените значение API_KEY на ваш полученный ключ.
Как запустить
Склонируйте репозиторий или скачайте файлы.
Убедитесь, что установлены необходимые библиотеки (pip install requests).
Запустите файл main.py (или другое имя файла с вашим кодом).
Пример использования
Выберите валюту "из" и "в".
Введите сумму.
Нажмите кнопку "Конвертировать".
История операций отобразится ниже в таблице.
Замечания
История сохраняется в файл history.json.
При повторном запуске история восстанавливается.
